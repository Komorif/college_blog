<mark style="background: #FF5582A6;">Множества в Python</mark> — это неупорядоченные коллекции **уникальных** элементов.

<mark style="background: #FF5582A6;">1. Основные свойства</mark>

- **Уникальность** — нет дубликатов.
- **Неупорядоченность** — элементы хранятся без порядка (как словари до Python 3.7).
- **Изменяемость** — можно добавлять и удалять элементы.
- **Элементы** должны быть **хешируемыми** (числа, строки, кортежи, но не списки или словари).

Создание множества

```python
my_set = {1, 2, 3} # через фигурные скобки
empty_set = set() # пустое множество (не '{}', это словарь)
```

<mark style="background: #FF5582A6;">2. Основные операции</mark>

(1) Добавление

```python
my_set.add(4) # {1, 2, 3, 4}
my_set.update([5, 6]) # {1, 2, 3, 4, 5, 6}
```

(2) Удаление

```python
my_set.remove(3) # удаляет элемент, ошибка если нет
my_set.discard(10) # удаляет, если есть (без ошибки)
```

(3) Проверка принадлежности:

```python
if 2 in my_set:
    print("Есть!")
```
  
<mark style="background: #FF5582A6;">3. Особые методы</mark>

- `len(my_set)` — количество элементов.
- `my_set.pop()` — удаляет и возвращает случайный элемент.
- `my_set.clear()` — очищает множество.
