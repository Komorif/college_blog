<mark style="background: #FF5582A6;">1. Базовые срезы</mark>

Срез с двумя параметрами

```python
s="0123456789"
s1=s[3:8] # "34567"
```

Срез без первого и последнего символа

```python
s="0123456789"
s1=s[1:-1] # "12345678"
```

Примеры срезов

```python
s="0123456789"
s1=s[:8] # "01234567" – от начала строки
s2=s[3:] # "3456789" – до конца строки
s3=s[::-1] # "9876543210" – реверс строки
```


Срезы с отрицательными индексами:

```python
s = "электроника"
s1= s[:-2] # электрони
s2 = s[-7:-3] # трон
```

<mark style="background: #FF5582A6;">(2) Удаление некоторых символов</mark>

```python
s = "электроника"
s1=s[:3]+s[7:] # эленика
s2= s[:3]+ "ктр"+s[8:] # электрика
```

Срез с тремя параметрами s[a:b:d] d – задает шаг

```python
s="0123456789"
s1=s[1:9:2] # "1357"
```

Вывод всех символов с индексами

```python
print(s[::2]) # все четные
print(s[1::2]) # все нечетные начиная со второго символа строки
print(s[::-2]) # все символы через один в обратном порядке, начиная с последнего
```
