<mark style="background: #FF5582A6;">Функция</mark> - отрезок кода, который можно вызвать через одну единственную строку многократно. Применение функций позволяет избавиться от повторений и упростить код.

Пример создания функции:

```python
def func_name(argument):
    body(инструкции)
    return [значение]
```

Функцию можно объявить где угодно. Если сделать это внутри класса, то она будет называться методом класса и вызываться так:

```python
class_name.function()
```

<mark style="background: #FF5582A6;">Пример:</mark> функция складывает два числа, переданные в качестве аргументов. Если один или оба аргумента не были переданы, используются значения по умолчанию.

```python
def print_sum(a=2, b=2):  
    sum = a+b  
    print(sum)  
    return # Вернёт None  
# чтобы вызвать функцию  
print_sum(x,y)
```

Значение функции можно сразу передать в другую функцию или переменную.

<mark style="background: #FF5582A6;">Пример возвращения списка:</mark>

```python
def x(n):  
    a = [1,3]  
    a = a * n  
    return a  
print(x(2)) # выведет [1,3,1,3]
```

<mark style="background: #FF5582A6;">Пример возвращения нескольких значений: </mark>

```python
def example():  
    language = "python"  
    version = 3  
    flag = True  
    return language, version, flag  
language, version, flag = example()  
print(language,version,flag) # будет выведено python 3 True
```
