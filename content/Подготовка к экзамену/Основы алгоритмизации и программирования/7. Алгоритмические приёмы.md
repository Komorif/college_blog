(1) Необходимо вычислить значение 1!+2!+…+n!

```python
n = int(input("Сколько факториалов будем суммировать? "))
s = 0 # начальное значение суммы
for i in range(1, n+1):
	# Вычисление факториала
	p = 1
for k in range(1, i+1):
	p = p * k
	# Добавление вычисленного факториала к сумме 
	s = s + p
print("Сумма факториалов =", s)
```

(3) Счетчик событий. Пользователь вводит 9 чисел, определить сколько из них являются одновременно четными и положительными

```python
count = 0 # Обнуляем переменную счетчик
for i in range(0, 9):
	x = int(input("Введите число: "))
	if (x % 2 == 0) and (x > 0):
		count = count + 1
print(count)
```

(4) Досрочное завершение цикла использование операторов break и continue

```python
for n in range(20):  
    if n % 2 == 0:  
        continue  
    if n == 17:  
        break  
  
    print(n)
```
