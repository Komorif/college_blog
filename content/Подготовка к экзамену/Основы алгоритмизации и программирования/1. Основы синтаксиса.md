<mark style="background: #FF5582A6;">1. Арифметические операции</mark>

| Операция     | Объяснение                                                                                                  |
| ------------ | ----------------------------------------------------------------------------------------------------------- |
| x+y          | Сложение                                                                                                    |
| x-y          | Вычитание                                                                                                   |
| x*y          | Умножение                                                                                                   |
| x/y          | Деление x на y<br>4 / 1.6 = 2.5<br>10 / 5 = 2.0                                                             |
| x//y         | Деление x на y нацело<br>11 // 4 = 2<br>11.8 // 4 = 2.0<br>Результат целый, только если оба аргумента целые |
| x%y          | Остаток от деления<br>11 % 4 = 3                                                                            |
| x**y         | Возведение x в степень y<br>2**3=8                                                                          |
| abs(x)       | Модуль числа x                                                                                              |
| round(x)     | Округление<br>round(11.3)=11                                                                                |
| round(x, n)  | Округляет число x до n знаков после запятой<br>round(12.345673)=12.346                                      |
| pow(x, y)    | Полный аналог записи x**y                                                                                   |
| divmod(x, y) | Выдает 2 числа: частное и остаток, обращаться следует так:<br>q, r = divmod (x, y)                          |

<mark style="background: #FF5582A6;">2. Встроенные функции</mark>

| Функция        | Объяснение                                                |
| -------------- | --------------------------------------------------------- |
| max(a, b, ...) | Максимальное число из 2-х или более<br>max([2,6,3])=6     |
| min(a, b, ...) | Минимальное число из 2-х или более<br>min([2,6,3])=2      |
| max(seq)       | Максимальный элемент последовательности<br>max([2,6,3])=6 |
| min(seq)       | Минимальный элемент последовательности<br>min([2,6,3])=2  |
| sum(seq)       | Сумма элементов последовательности<br>sum((2,6,3))=11     |
| sorted(seq)    | Отсортированный список<br>sorted([3,2,5,1,4])=[1,2,3,4,5] |
