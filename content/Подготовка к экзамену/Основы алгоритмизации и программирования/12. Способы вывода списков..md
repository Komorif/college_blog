<mark style="background: #00B7EB;">1. С помощью print()</mark>

```python
b = [17, 409, 88]
print(b) # [17, 409, 88]
```

<mark style="background: #00B7EB;">2. Вывод каждого элемента списка по-отдельности</mark>

```python
a = ["Андрей", "Вера", "Даша", "Коля", "Юра"]
for i in range(5):
	print(a[i])
```

<mark style="background: #00B7EB;">3. Вывод каждого элемента списка по-отдельности в одной строке</mark>

```python
a = ["Андрей", "Вера", "Даша", "Коля", "Юра"]
for i in range(5):
	print(a[i], end=" ")
# Андрей Вера Даша Коля Юра 
```

<mark style="background: #00B7EB;">4. Вывод элементов списка без обращения к индексам элементов</mark>

```python
fruits = ["Яблоко", "Банан", "Груша"]
for x in fruits:
	print(x, end=" ")
```
