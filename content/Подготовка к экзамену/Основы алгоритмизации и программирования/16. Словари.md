<mark style="background: #FF5582A6;">Что такое словарь(dict)?</mark>

- **Изменяемая** коллекция пар **ключ-значение**.
- Ключи должны быть **уникальными** и **неизменяемыми** (числа, строки, кортежи).
- Значения могут быть любого типа (даже списки или другие словари).
- Неупорядоченные (до Python 3.7), но после Python 3.7+ сохраняют порядок добавления.

<mark style="background: #FF5582A6;">1. Создание словаря</mark>

```python
# Способ 1: Через фигурные скобки
my_dict = {"name": "Alice", "age": 25}
# Способ 2: Через dict()
empty_dict = dict()  
from_list = dict([("key1", 1), ("key2", 2)])  # {"key1": 1, "key2": 2}
# Способ 3: Через dict comprehension
squares = {x: x**2 for x in range(5)}  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
```

<mark style="background: #FF5582A6;">2. Основные операции</mark>

(1) Добавление/изменение элементов:

```python
my_dict["email"] = "alice@example.com"  # Добавляет или обновляет
```

(2) Получение значения:

```python
name = my_dict["name"]         # Ошибка, если ключа нет
age = my_dict.get("age", 0)    # Возвращает 0, если ключа нет (без ошибки)
```

(3) Удаление элементов:

```python
del my_dict["age"]             # Удаляет ключ "age"
value = my_dict.pop("name")    # Удаляет и возвращает значение
my_dict.clear()                # Очищает словарь
```


<mark style="background: #FF5582A6;">3. Полезные методы</mark>

(1) Проверка ключей:

```python
if "name" in my_dict:  
    print("Ключ существует!")
```

(2) Получение всех ключей/значений:

```python
keys = my_dict.keys()    # ["name", "age"]
values = my_dict.values()  # ["Alice", 25]
items = my_dict.items()   # [("name", "Alice"), ("age", 25)]
```

(3) Обновление словаря:

```python
my_dict.update({"age": 26, "city": "Moscow"})  # Добавляет или меняет значения
```


<mark style="background: #FF5582A6;">4. Примеры использования</mark>

(1) Подсчёт частоты элементов в списке:

```python
words = ["apple", "banana", "apple", "cherry"]  
freq = {}  
for word in words:  
    freq[word] = freq.get(word, 0) + 1  
# Результат: {"apple": 2, "banana": 1, "cherry": 1}
```

(2) Хранение данных в структурированном виде:

```python
person = {  
    "name": "Alice",  
    "age": 25,  
    "skills": ["Python", "SQL"]  
}
```
