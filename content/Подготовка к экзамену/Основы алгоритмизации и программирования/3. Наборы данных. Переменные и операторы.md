<mark style="background: #FF5582A6;">1. Наборы данных</mark>

(1) integer - целые числа (0, 12, 999999)

```python
x = 10
print(type(x))  # <class 'int'>
```

(2) float - дробное число (0.5, 3.4, 10.0)

```python
y = 3.14
print(type(y))  # <class 'float'>
```

(3) string - строки ("строка", 'кавычка', "456")

```python
name = "Привет"
print(type(name))  # <class 'str'>
```

(4) boolean - логический тип данных (правда(True) - 1, ложь(False) - 0)

```python
is_active = True
print(type(is_active))  # <class 'bool'>
```

(5) list - список (хранит объекты)
<font style="color: red;">[</font>1, "строка", True, 0.0<font style="color: red;">]</font> <-- его <font style="color: red;">можно изменять</font> (добавлять, удалять, менять элементы)

```python
my_list = [1, "текст", True]
my_list.append(42)
print(my_list)  # [1, 'текст', True, 42]
```

(6) tuple - кортеж (хранит объекты)
<font style="color: red;">(</font>4, "тест", 9.34<font style="color: red;">)</font> - <font style="color: red;">неизменяемый</font> (только преобразовать в массив и затем изменить)

```python
my_tuple = (1, "строка", False)
print(my_tuple[0])  # 1
# my_tuple[0] = 2  # ❌ Ошибка! Кортеж нельзя изменить
```

(7) set - множество (хранит объекты) (у элементов нет номеров) 
{4, 3, 'стрк'}
<font style="color: red;">повторяющиеся элементы удаляются,</font> изменяемый тип (добавлять/удалять элементы можно) 

```python
my_set = {1, 2, 2, 3}
print(my_set)  # {1, 2, 3}
my_set.add(4)
print(my_set)  # {1, 2, 3, 4}
```

(8) dict - словарь {"картина": "изображение на холсте"}

```python
my_dict = {"яблоко": "фрукт", 1: [1, 2, 3]}
print(my_dict["яблоко"])  # фрукт
```

<mark style="background: #FF5582A6;">2. Переменные</mark>

Можно использовать в названии переменной:
1. Можно использовать только Латинские буквы(A-Z, a-z)
2. Русские буквы не рекомендуется
3. имя не может начинаться с цифры
4. знак подчеркивания __

Нельзя использовать в названии переменной:
1. скобки
2. знаки +, =, !, ? и др.

<mark style="background: #FF5582A6;">3. Операторы</mark>


<mark style="background: #00B7EB;">3.1. Операторы сравнения</mark>

1. == - если равны оба операнда, то условие становится истинным
2. := или <> - если не равны оба операнда, то условие становится истинным >, <, >=, <=

<mark style="background: #00B7EB;">3.2. Операторы присваивания</mark>

| Оператор | Примеры                                           |
| :------- | ------------------------------------------------- |
| +=       | c=5 a=2<br>c+=a <=> c=c+a<br>Значение: c=7        |
| -=       | c=5 a=2<br>c-=a <=> c=c-a<br>Значение: c=3        |
| *=       | c=5 a=2<br>c*=a <=> c=c ***** a<br>Значение: c=10 |
| /=       | c=5 a=2<br>c/=a <=> c=c/a<br>Значение: c=5        |
| %=       | c=5 a=2<br>c%=a <=> c=c%a<br>Значение: c=1        |
| **=      | c=5 a=2<br>c**=a <=> c=c****a<br>Значение: c=9    |
| //=      | c=5 a=2<br>c//=a <=> c=c//a<br>Значение: c=5      |
