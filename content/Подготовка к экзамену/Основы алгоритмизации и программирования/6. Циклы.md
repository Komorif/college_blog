<mark style="background: #FF5582A6;">1. Цикл For</mark>

For - счётный цикл (мы заранее знаем количество повторений)

(1) Перебор строки

```python
word_str = "Hello, world!"  
for n in word_str:  
    print(n) # Выведится по одной букве на каждой строчке
```

(2) Range

```python
# Пример 1
for название_переменной in range(количество_повторений):
	блок_кода

# Пример 2
for k in range(1, 11): # Диапазон от 1 до 10
	print(2**k)
# Вывод: 2 4 8 16 32 64 128 256 512 1024

# Пример 3
# k меняется от 10 до 0, уменьшаем каждый раз на 1 (т.е. k = 10 9 8 ...)
for k in range(10, 0, -1): 
    print(k**2)
# Вывод: 100 81 64 49 36 25 16 9 4 1

# range(начало, конец, шаг) 

# Пример 4
# Перебор через len
word = "hello world"  
for i in range(len(word)):  # len() - узнать длину строки
    print(word[i], end=' | ')
    
# h | e | l | l | o |   | w | o | r | l | d | 
```


<mark style="background: #FF5582A6;">2. Цикл While</mark>

While - условный цикл (количество повторений не знаем, но знаем условие, согласно которому нужно что-либо повторять)

```python
while условие:  # while - пока
   блок_кода   # выполняется пока выполняется условие (пока оно истино(True))
```

(1) В цикле запрашивать числа до тех пор, пока не введён ноль. Проверить, есть ли среди введенных чисел, число n.

```python
# запрос чисел до тех пор, пока не введен ноль
# проверка встретилось ли число n
n = int(input())
a = int(input())
while a != 0:
	if a == n:
		print("Встретилось число: ", n)
	a = int(input())
else:
	print("число не встретилось")
```

(2) Реализация бесконечного цикла

```python
# 1 пример
while True:  # бесконечный цикл
	if условие_1:  # условие для остановки цикла
	break

# 2 пример
while True:
	print("Введите положительное число: ")
	n=int(input())
	if n > 0:
		break
```

(3) Определить количество цифр в десятичной записи целого положительного числа, записанного в переменную n.

```python
n=int(input('Введите число: '))  
count=0  
while n>0:  
    count += 1  
    n = n // 10  
print('Количество цифр: ', count)
```
