<mark style="background: #FF5582A6;">2. Генератор случайных чисел</mark>

Для использования необходимо прописать: import random

```python
number = random.random() # записывает любое рандомное число
number = random.randint(20, 35) # записывает рандомное число от 20 до 35
```

```python
import random
array = [random.randint(1, 100) for i in range(10)]
for num in array:
    print(num, end=" ")
```
