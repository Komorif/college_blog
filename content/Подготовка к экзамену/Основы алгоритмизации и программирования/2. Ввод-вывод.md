<mark style="background: #FF5582A6;">1. Ввод-вывод</mark>

<mark style="background: #00B7EB;">1.1. Ввод</mark>

Функция input() считывает строку с клавиатуры и возвращает ее значение, которое сразу же можно присвоить переменным:

```python
a = input() # для текста
a = int(input()) # для целых значений
```

Если значения переменных заданы в одной строке используем метод <font style="color: red;">split()</font>

```python
a, b = input().split() # для текста
a, b, c = map(int, input().split()) # для целых значений
# map - применяет к каждому элементу списка заданную ф-ю
```

<mark style="background: #00B7EB;">1.2. Вывод</mark>

Параметр sep в функции print() задаёт **разделитель**, который будет вставляться **между выводимыми значениями**. По умолчанию это пробел.

Например, чтобы выводить значения через двоеточие ":", укажем параметр sep=":"

```python
print(a, b, c, sep=":")
```

Если вы хотите, чтобы значения выводились **без какого-либо разделителя**, передайте sep пустую строку:

```python
a = 1
b = 2
print(a, "+", b, "=", a+b, sep="") # 1+2=3
```

Чтобы вывести каждое значение с **новой строки**, используйте sep="\n":

```python
print(a, b, sep="\n")
```

А вот параметр end определяет, **что будет напечатано после всех значений**, указанных в print(). По умолчанию это перевод строки (end="\n"), но его можно изменить

```python
print(a, b, c, sep="", end="")
```

<mark style="background: #FF5582A6;">2. Форматный вывод</mark>

Использование f строк

```python
year = 2025
event = "сессии"
print(f"Результаты {event} {year} года")
```

Для строк операция S * n выполнит повторение строки S n-раз

```python
print(str(2+2)*int("2"+"2")) # выведет символ 4, повторенный 22 раза
```

<mark style="background: #00B7EB;">2.1. Форматный вывод вещественных чисел
</mark>

Целая и дробная части числа разделяются точкой

```python
x = 123.456
print("{:10.2f}".format(x)) # Вывод:     123.46

# 10 - всего знаков
# 2 - округление до двух знаков
# f - в дробной части
```

<mark style="background: #00B7EB;">2.2. Форматный вывод целых чисел</mark>

Пример 1

```python
a = 1  
b = 2  
print("{:d}+{:d}={:d}".format(a, b, a+b)) # 1+2=3

# {:d} - целые числа
```

Пример 2

```python
a = 5
print("{:5d}{:5d}{:5d}".format(a, a*a, a*a*a)) #    5   25  125

# :5d - всего 5 знаков
# т.е. a = 5 это 1 элемент, а значит остальные 4 элемента это пробелы
# далее a*a=25 => уже 2 элемента (значит перед 25 будет уже 3 пробела) и т.д.
```
