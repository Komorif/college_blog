Метод 1. <mark style="background: #FFB86CA6;">insert</mark> можно использовать, чтобы вставить элемент в список:

```python
a = [1, 2, 3, 4, 5]
a.insert (1, 'Привет')
print(a)
```

Метод 2. Можно добавлять элемент в конец списка. Для этого используется метод <mark style="background: #FFB86CA6;">a.append(x)</mark>

```python
a = [1, 2, 3]
a.append(4)
print(a) # [1, 2, 3, 4]
```
Метод 3. Можно расширять список, добавляя в его конец элементы другого списка. Для этого используется метод <mark style="background: #FFB86CA6;">a.extend(b):</mark>

```python
a = [1, 2, 3]
b = [4, 5]
a.extend(b)
print(a) # [1, 2, 3, 4, 5]
```

 Метод 4. Функция нахождения длины списка <mark style="background: #FFB86CA6;">len(a) </mark>

```python
a =["Яблоко", "Банан" ,"Груша"]
x = len(a)
print(x) # 3
```
  
Метод 5. Перестановка элементы списка в обратном порядке. Метод <mark style="background: #FFB86CA6;">a.reverse()</mark>:

```python
a = [0, 1, 2, 3, 4, 5]
a.reverse()
print(a) # [5, 4, 3, 2, 1, 0]
```

Метод 6. Удалить элемент списка можно с помощью метода <mark style="background: #FFB86CA6;">pop()</mark>, указав индекс элемента:

```python
a = [0, 1, 2, 3, 4, 5]
a.pop(2)
print(a) # [0, 1, 3, 4, 5]
```

Метод 7. Сортировка списка. Функция <mark style="background: #FFB86CA6;">sorted(a)</mark>:

```python
# по возрастанию
animals = ["кот", "еж", "собака", "барсук"]
animals = sorted(animals)
print(animals) # ["барсук", "еж", "кот", "собака"]


# по убыванию
a = [5, 65, 14, 700, 8]
a = sorted(a, reverse = True)
print(a) # [700, 65, 14, 8, 5]
```

Метод 8. В Python есть некоторые <mark style="background: #FFB86CA6;">агрегатные функции</mark>

```python
a = [5, 4, 3, 2, 1]
print(min(a)) # 1
print(max(a)) # 5
print(sum(a)) # 15
```

 Пример генерации списка случайными числами:

```python
# 10 чисел, сгенерированных случайным образом в диапазоне (1, 100)
import random
l = [random.randint(1,100) for x in range (10)]
print(l) # [44, 61, 66, 90, 46, 13, 55, 9, 69, 27]
```
