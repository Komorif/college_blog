<mark style="background: #FF5582A6;">1. Форматирование строк - метод Format</mark>

<mark style="background: #00B7EB;">1.1. Одна подстановка</mark>

```python
'Hello, {}!'.format('Vasya') # Hello, Vasya!
```

<mark style="background: #00B7EB;">1.2. Несколько подстановок</mark>

```python
'{0}{1}{0}'.format('abra', 'cad') # abracadabra
```

<mark style="background: #00B7EB;">1.3. Множественные подстановки</mark>

```python
'коориданты: {широта}, {долгота}'.format(широта='37.24c', долгота='115.813')
# коориданты: 37.24c, 115.813
```

<mark style="background: #FF5582A6;">2. Методы Boolean</mark>

Методы Boolean - предоставляют результат в виде логического значения (true/fal

| Метод           | Значение                                             |
| --------------- | ---------------------------------------------------- |
| str.isalnum()   | Строка состоит только из алфавитно-цифровых символов |
| str.isalpha()   | Строка состоит только из букв                        |
| str.lower()     | Строка содержит буквы только в нижнем регистре       |
| str.upper()     | Строка содержит буквы только в верхнем регистре      |
| str.isnumeric() | Строка состоит только из числовых символов           |
| str.istitle()   | Слова в строке начинаются с заглавной буквы          |

<mark style="background: #FF5582A6;">3. Поиск в строке</mark>

| Метод и форма         | Значение                                                                    |
| --------------------- | --------------------------------------------------------------------------- |
| find()                | Возвращает индекс первого вхождения подстроки в строку и имеет три формы    |
| find(str)             | Поиск подстроки str ведется с начала строки до ее конца                     |
| find(str, start)      | Параметр start задает начальный индекс, с которого будет производится поиск |
| find(str, start, end) | Параметр end задает конечный индекс, до которого будет идти поиск           |
Если подстрока не найдена, метод возвращает -1

<mark style="background: #FF5582A6;">4. Замена в строке</mark>

| Метод и форма          | Значение                                                                     |
| ---------------------- | ---------------------------------------------------------------------------- |
| replace(old, new)      | Заменяет подстроку old на new                                                |
| replace(old, new, num) | параметр num указывает, сколько вхождений подстроки old надо заменить на new |

```python
s1 = "breHeHeHeHs"
s3 = s1.replace("He", "ХоХо", 2)
print(s3) # breХоХоХоХоHeHs
```

<mark style="background: #FF5582A6;">5. Разделение на подстроки</mark>

Метод split() - разбивает строку на список подстрок (в зависимости от разделителя) 

| Метод и форма   | Значение                                                                  |
| --------------- | ------------------------------------------------------------------------- |
| split()         | в качестве разделителя используется пробел                                |
| split(del)      | в качестве разделителя используется del                                   |
| split(del, num) | параметр num указывает, сколько вхождений del используется для разделения |

<mark style="background: #FF5582A6;">6. Соединение строк</mark>

Метод join() - слияние 2-х строк

```python
s = "hello"
s1 = "-".join(s)
print(s1) # h-e-l-l-o

# Вывод в обратном порядке  
s = "Весна, пришла!"  
print("".join(reversed(s)))
```

<mark style="background: #FF5582A6;">7. Подсчет количества вхождений одной строки в другую</mark>

Метод count() - подсчитывает количество вхождений подстроки s в строку s1

Результатом является число. Можно указать позицию начала поиска i и окончания поиска j:

```python
s1 = "abrakadabra"
print(s1.count("ab")) # 2
print(s1.count("ab", 1)) # 1
```
