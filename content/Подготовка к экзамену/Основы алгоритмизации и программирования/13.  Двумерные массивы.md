<mark style="background: #FF5582A6;">Двумерные массивы</mark> - прямоугольные таблицы с данными

Создать числовую таблицу из 2 строк и 3 столбца можно так:

```python
A = [[1, 2, 3], [4, 5, 6]]
```

Здесь первая строка списка A[0] является списком из чисел [1, 2, 3]

То есть

```python
A [0][0] == 1
A [0][1] == 2
A [0][2] == 3
A [1][0] == 4
A [1][1] == 5
A [1][2] == 6
```

  
<mark style="background: #FF5582A6;">1. Вывод двумерного массива</mark> 

Для обработки и вывода списка, как правило, используется 2 вложенных цикла. Первый цикл - по номеру строки, второй цикл - по элементам внутри строки.

Например, вывести двумерный числовой список на экран построчно можно так:

```python
A = [[1, 2, 3], [4, 5, 6]]
for i in range(len(A)):  
    for j in range(len(A[i])):  
        print(A[i][j], end=' ')  
    print()  
# вывод  
# 1 2 3  
# 4 5 6
```

Другой способ, но циклы не по индексу, а по значениям списка:

```python
A = [[1, 2, 3], [4, 5, 6]]  
for stroka in A:  
    for elem in stroka:  
        print(elem, end=' ')  
print()  
#вывод тот же
```

<mark style="background: #FF5582A6;">2. Примеры</mark> 

Пример 1. Используем два вложенных цикла

```python
A = [[1, 2, 3], [4, 5, 6]]  
S = 0  
for i in range(len(A)):  
    for j in range(len(A[i])):  
        S += A[i][j]  
print("сумма = ", S)  
#вывод: сумма = 21
```

Пример 2. Вычисление суммы всех чисел в списке (метод по значению строк)

```python
A = [[1, 2, 3], [4, 5, 6]]  
S = 0  
for stroka in A:  
    for elem in stroka:  
        S += elem  
	print("сумма = ", S)  
#вывод: сумма = 21
```

<mark style="background: #FF5582A6;">3. Создание вложенных списков</mark> 

(1) Сначала создадим список из n элементов. Затем сделаем каждый элемент списка ссылкой на другой одномерный список из m элементов:

```python
n = 5  
m = 5  
A = [0] * n  
for i in range(n):  
    A[i] = [0] * m
```

(2) Создадим пустой список, потом n раз добавим в него новый элемент, являющийся списком - строкой:

```python
n = 5  
m = 5  
A = []  
for i in range(n):  
    A.append([0]*m)
```

(3) Можно воспользоваться генератором: создаём список из n элементом, каждый из которых будет списком, состоящим из m нулей:

```python
n = 5
m = 5
A = [[0]*m for i in range(n)]
```

  
<mark style="background: #FF5582A6;">4. Ввод двумерного массива</mark> 

```python
n = 3  
m = 3  
A = []  
for i in range(n):  
    A.append(list(map(int, input().split())))  
print(A)
```

При помощи генератора

```python
n = 3
m = 3
A = [list(map(int, input().split())) for i in range(n)]
```

<mark style="background: #FF5582A6;">5. Обработка двумерного массива</mark> 

Дан квадратный массив из n строк и n столбцов, т.е. A[i]  [j], для которых i = j.

Элементы, которые лежал выше главной диагонали это элементы A [i]  [j], для которых i < j, а ниже главной диагонали i > j.

Ввести матрицу 3х3. Переставить 1-й и 3-й столбцы матрицы. Вывести полученную матрицу на экран.

```python
n = 3  
m = 3  
A = []  
for i in range(n):  
    A.append(list(map(int, input().split())))  
for i in range(n):  
    c = A[i][0]  
    A [i][0] = A [i][2]  
    A [i][2] = c  
for i in range(n):  
    for j in range(m):  
        print(A[i][j], end=' ')
    print()
```
