Синтаксис

Общий синтаксис конструкции if-elif-else:

```python
if условие1:
    блок_кода1

elif условие2:
    блок_кода2

elif условие3:
    блок_кода3

else:
    блок_кодаN
```

Условия проверяются по порядку. Если какое-то из них истинно, выполняется соответствующий блок кода, а остальные игнорируются. Если ни одно из условий не истинно, выполняется блок кода в разделе else, если он присутствует.

Пример 1. Составить программу для решения уравнения ax^2+bx+c=0. Формат входных данных: значения коэффициентов a, b и c вводятся с клавиатуры через пробел в одной строке.

```python
from math import sqrt  
print("Введите a,b,c:")  
a,b,c = map(float, input().split())  
p = b*b-4*a*c  
if p < 0:  
    print("Нет корней")  
else:  
    x1=(-b+sqrt(p))/(2*a)  
    x2=(-b-sqrt(p))/(2*a)  
print("x1={:5.3f} x2={:5.3f}".format(x1, x2))
```

Пример 2. В переменных a и b записаны возрасты Андрея и Бориса. Кто из них старше?

```python
if a > b:
	print("Андрей старше")
else:
	if a == b:
		print("Одного возраста")
	else:
		print("Борис старше")
```

Каскадное ветвление

```python
if a > b:
	print("Андрей старше")
elif a == b:
	print("Одного возраста")
else:
	print("Борис старше")
```

Пример 3. Вычислить размер скидки

```python
cost = 1500
if cost < 1000:
	print("Скидок нет")
elif cost < 2000:
	print("Скидка 2%")
elif cost < 5000:
	print("Скидка 5%")
else:
	print("Скидка 10%")

# Скидка 2%
```

Пример 4. Набор сотрудников в возрасте 25-40 лет (включительно)

```python
if v >= 25 and v <= 40:
	print("подходит")
else:
	print("не подходит")
```

Приоритет
1. Отношения (<, >, <=, >=, !=)
