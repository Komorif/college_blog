<mark style="background: #FF5582A6;">1. Импорт базы данных из "эксель таблицы"</mark>

(1) Кликаем правой кнопкой мыши по БД deuser далее: Tasks / Import data...

![[Pasted image 20250522203751.png]]

(2) В Data source указываем Microsoft Excel. В Excel file path файл типа "xlsx" с уже **готовой** БД. Excel version можно выбрать последнюю

![[Pasted image 20250522203932.png]]

(3) Нажимаем Use SQL Server Authentication. Вводим имя и пароль

![[Pasted image 20250522205407.png]]

(4) Здесь нажимаем copy data (то есть копируем все)

![[Pasted image 20250522205521.png]]

(5) Выделяем все таблички

![[Pasted image 20250522205538.png]]

(6) Тут просто жмем next

![[Pasted image 20250522205603.png]]

(7) И finish

![[Pasted image 20250522205615.png]]

<mark style="background: #FF5582A6;">2. Создание связей</mark>

(1) Жмем правой кнопкой мыши по Database Diagrams. Затем New Database Diagram

![[Pasted image 20250522210725.png]]

(2) В окошке Add table добавляем все таблицы (если это нужно)

![[Pasted image 20250522210821.png]]

(3) Первым делом устанавливаем primary keys

![[Pasted image 20250522211102.png]]

(4) Нажимаем на названии таблички правой кнопкой мыши затем: Table View / Standart

![[Pasted image 20250522213450.png]]

(5) Настраиваем все в соответствии с заданием
Data Type - тип данных
Allow Nulls - галочки нет (NOT NULL) / галочка есть (NULL)

![[Pasted image 20250522213550.png]]

(6) Создаем связи путем протягивания от одной строчке к другой

![[Pasted image 20250522215301.png]]

(7) Нажимаем правой кнопкой мыши по диаграмме и сохраняем
