<mark style="background: #FF5582A6;">1. Простой вариант: репозиторий на GitHub при помощи GitHub Desktop</mark>

(1) Переходим по [ссылке](https://desktop.github.com/download)и скачиваем GitHub Desktop

(2) При первом запуске появится приветственное окно. Нажимаем Sign in to GitHub.com

![[Pasted image 20250529203353.png]]

(3) Выполняем вход в свой аккаунт (если нет аккаунта регистрируемся)

![[Pasted image 20250529203509.png]]

(4) После входа выполняем авторизацию - нажимаем Authorize desktop

![[Pasted image 20250529212031.png]]

(5) После авторизации нас перекинет в приложение здесь жмем Finish

![[Pasted image 20250529212535.png]]

(6) После всех проделанных шагов вы попадаете в окно Let's get started

![[Pasted image 20250529204330.png]]

(7) Переходим в профиль затем жмем Repositories, после чего создаем новый репозиторий при помощи кнопки New

![[Pasted image 20250529204858.png]]

(8) В Repository name пишем имя своего репозитория, по желанию можно добавить описание (Description). Ставим галочку на Public (чтобы сделать наш репозиторий просматриваемым для всех). Остальное можем не трогать. Нажимаем зеленую кнопку Create repository

![[Pasted image 20250529205109.png]]

(9) Здесь нам нужно скопировать ссылку на наш репозиторий

![[Pasted image 20250529205706.png]]

(10) Переходим обратно в GitHub Desktop. Жмем на кнопку Clone a Repository from the Internet from the Internet...

![[Pasted image 20250529210122.png]]

(11) Нажимаем URL и вставляем ссылку из (9) пункта. Local Path это то место где будет храниться наш репозиторий локально (запоминаем путь до нег), затем жмем clone

![[Pasted image 20250529210314.png]]

(12) Переходим по пути до нашего репозитория. После чего можем изменять и добавлять файлы в проект. (Для примера на картинке ниже в репозитории создан .txt файл). 

![[Pasted image 20250529210559.png]]

(13) Переходим в GitHub Desktop (видим что в репозитории появились изменения). Затем как угодно называем наш комит (test commit), затем нажимаем commit

![[Pasted image 20250529210905.png]]

(14) После прожатия кнопки commit, нажимаем push origin

![[Pasted image 20250529211409.png]]

(15) Затем проверяем наш репозиторий
